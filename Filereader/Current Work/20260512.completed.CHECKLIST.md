# Fix Initialization Crash and Audit

## Aim and Metadata
- Aim: Fix the initialization crash caused by calling BeginInvoke before the window handle is created, and perform a codebase audit.
- Creation date: 2026-05-11
- Last updated: 2026-05-11
- Scope target: `MainForm.cs`, `logs/`

  - Evidence: Added `IsHandleCreated` guard to `ItemCheck` (line ~464) and moved Startup Toast to `Load` event (line ~73).
